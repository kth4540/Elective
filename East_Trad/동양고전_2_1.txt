##한자의 분류-육서

	상형

		일종의 그림문자로 사물의 모양을 본떠 만든 글자, 초기 한자는 상형에서 시작

		해 일, 달 월 , 입 구, 귀 이

	지사

		일종의 기호로 추상적인 개념은 약속된 기호로 표시함

		위 상, 아래 하, 가운데 중, 뿌리 본

	먼저 만들어진 한자를 조합하여 새로운 글자를 만들게됨

	형성

		조합하는 방식에서 의미를 나타내는 글자와 발음을 나타내는 글자를 조합

		삼수변 + ...

	회의

		둘 이상의 의미가 합쳐져 새로운 뜻을 갖게 된 것

		밝을 명-> 해 일 + 달 월(밝은 것 2개)

		믿을 신 -> 사람 인 + 말씀 언(사람의 말은 믿을만해야한다)

		쉴 휴 -> 사람 인 + 나무 목(사람이 나무에 기대어 쉰다)

	전주

		원래 존재하는 글자의 의미를 확대,변화시켜 새로운 뜻으로 바뀜

		풍류 악->즐거울 락, 종아할 요

		악할 악-> 미워할 오

		말씀설 -> 달랠 세 , 기쁠 열

	가차

		새로운 사물이나 외래어를 한자의 의미나 소리를 이용해 표기

		음차- 소리를 이용(영국,코카콜라)

		훈차- 의미를 이용(컴퓨터->전자 뇌)

##한자쓰기

	놈 자-> 기자, 환자, 석자

	반드시 필 -> 필수, 생필품, 필시

	다닐 행 -> 행위, 행적, 행방

	굳셀 과 -> 과단, 인과

	생각할 유 -> 사유

	옳을 의 -> 의무, 강의

	바 소 -> 소문, 숙소, 소용

	있을 재 -> 재학, 재고, 부재

	붉을 적 -> 적자(손해) , 적자(갓 태어난 아이), 적색

	잃을 실 -> 실례, 실패, 실직

## 맹자 이루장구 하

	대인자, 언불필신 -> 대인이란 반드시 말이 믿을만 한 가를 기대하지 않고

	행불필과, 유의소재 -> 행동이 과단성 있기를 요구하지 않으며 오직 옳은 일을 행한다

	대인자, 불실기적자지심자야 -> 대인이란 어린아이의 마음을 잃지 않은 사람이다

##맹자

	맹자는 기원전 371-298에 태어나 살다간 사람

	성인이 인의로 다스리는 나라를 이상적 형태로 여김

	맹자가 활동하던 시대에는 패권주의자들이 많아 등용되지 않음

	은퇴 후 시경과 서경을 재해석, 맹자를 저술

	원래 대인이나 군자는 봉건시대때 지배층을 뜻하는 용어

	맹자는 인,의를 갖춘 사람으로 재규정(대장부, 호연지기,대인등이 맹자가 강조한 용어)

	의를 꾸준히 실천하면 호연지기가 길러지고, 호연지기를 갖춘 사람이 대장부
